Срок: 1-2 дня

Требуется написание небольшой системы по сбору и обработке данных.

Суть задачи: 

Есть публичная страница (по адресу website.com) которая содержит форму с несколькими вопросами:

1. Сколько цветов в радуге?
Варианты ответов (checkbox): 2, 4, 14

2. Сколько дней в году?
Варианты ответов (input): пользователь вводит ответ самостоятельно

3. Кто проживает на дне океана?
Варианты ответов (select): Спанч боб, Медуза, Русалка.

+ кнопка отправки результатов.

Заполнить таблицу может любой пользователь, авторизация не требуется.
Отправка ответов должна происходить без перезагрузки страницы.
В случае частичного заполнения формы и перезагрузки страницы, результаты не должны быть утеряны, после перезагрузки пользователь может продолжить с того места, на котором остановился.
Форму можно заполнить только 1 раз. После отправления результатом теста появляется всплывающее окно с текстом "Спасибо, ваши результаты отправлены".
Заполнить форму еще раз можно только после очистки cookies в браузере.




Есть страница администратора (website.com/admin)

На данной странице администратор может видеть отчеты в нескольких форматах:

Таблица с колонками (IP, Результат 1 вопроса, Результат 2 вопроса, Результат 3 вопроса, Количество отправок теста).
В случае если пользователь с одним и тем же IP отправил результаты несколько раз - высчитываем среднее арифметическое.
Порядок строк в таблице можно сортировать нажав на th в thead таблицы, в этом случает строки сортируются в соответствии с выбранной колонкой.
Можно удалять отдельные строки таблицы.

Результаты из таблицы можно экспортировать в CSV.


Под таблицей выводим круговую диаграмму из 3 секторов. Каждый сектор - это процентное отношение правильных ответов на каждый вопрос по отношению в другим вопросам.


Все собираем на верстке и дизайне bootstrap 3.
